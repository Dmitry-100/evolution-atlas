# Промпты для иллюстраций глобальных вымираний

Эти промпты рассчитаны на вертикальные изображения только для страницы вымираний. Сохраняйте результат как JPG в `public/assets/images/extinctions/portrait/` с указанным именем и затем обновляйте AVIF через `node scripts/optimize-images.mjs`.

Общий стиль для всех сцен: вертикальный 9:16 portrait, phone-ready paleoart poster, кинематографичная документальная реконструкция, низкая камера у земли или берега. Нижняя треть кадра должна показывать следы вымирания флоры и фауны крупным детальным передним планом: погибшие или выброшенные организмы, сломанные растения, раковины, кости, рифовые обломки, пепел, грязь, лед или загрязнение. Верхняя треть и дальний план должны показывать артефакты самой катастрофы: ледник, аноксичное море, вулканы, лавовые трещины, ударные пожары, дым, индустриальный горизонт. Средняя часть связывает передний план и дальний план атмосферой, туманом, водой, пеплом или береговой линией. Палитра приглушённая: холодные сине-серые и графитовые тона, редкие янтарные или вулканические акценты. Без текста, без схем, без логотипов, без инфографики, без людей, без сенсационной крови.
