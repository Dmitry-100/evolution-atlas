## 6. Сократические ответы — `buildTourRoute.ts` (`revealFor`)

**Главное (P1):** в строке 850 убрать обёртку. Сейчас:
```
return `Интересный факт: ${note} Важно держать рядом два масштаба: конкретную черту на
этой остановке и длинную линию наследования, где эта черта получает новый смысл.`;
```
Предлагается: `return note;` — `note` уже содержит законченную мысль. Хвост дословно повторяется
на каждой остановке и сильнее всего читается как «ИИ».

То же в `childRevealFor:1006` — вернуть `note` без хвоста «Интересный факт для прогулки…».

Сами `note` в основном хорошие. Точечно:

## 8. Вне тура

**`AboutPage.tsx`** — тезис «не X, а Y» стоит дважды подряд («Дерево родства вместо лестницы»,
«Мы не произошли от современных обезьян»). Это уместно как манифест, но второй раз можно
переформулировать прямо: «У нас с обезьянами общий предок; дальше каждая ветвь шла своим путём».

**`lineage.ts`** (`summaryRu`/`whyMattersRu`) — в целом чисто. Лёгкая частотность слов
«основа / будущий / задолго до» в `whyMattersRu`; не критично, точечно разнообразить при желании.
Менять массово не рекомендую — текст плотный и человеческий.
