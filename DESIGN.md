# Design System

## Direction

Премиальный темный музейный атлас: не landing page и не учебная таблица, а интерактивная экспозиция. Пользователь сразу видит масштаб времени, активный вид и объяснение, почему этот узел важен для нашей линии.

## Visual Principles

- Первый экран - сам атлас: левая навигация, центральная шкала времени, правая карточка экспоната.
- Глубокая шкала времени линейная, чтобы честно показать: почти вся история жизни прошла до появления приматов.
- Режим "Приматы крупно" использует отдельную портретную ось, где точки разведены ради чтения и выбора.
- Визуальная иерархия строится на контрасте: темный графитовый фон, теплые музейные акценты, приглушенные научные цвета эпох.
- Карточки используются только как функциональные панели: навигация, шкала, активный экспонат, источники.
- Никаких декоративных градиентных шаров. Фон - текстура, свет, слои и реальные/локальные изображения.

## Tokens

- `--bg`: почти черный графит.
- `--panel`: полупрозрачная музейная поверхность.
- `--text`: теплый белый.
- `--muted`: приглушенный бежево-серый для пояснений.
- `--gold` / `--amber`: активные элементы, даты, фокус.
- `--teal`, `--moss`: древняя жизнь, животные, переходы.
- Border radius: 8px максимум для рабочих поверхностей.

## Typography

- Заголовки: Georgia-like serif для музейного, книжного ощущения.
- Интерфейс: system sans для читаемости.
- Letter spacing в основном 0; увеличенная разрядка только для маленьких uppercase-лейблов.

## Interaction

- Hover по шкале времени меняет активный вид.
- Click/focus по точке делает то же самое.
- Slider работает по той же шкале, что и текущий визуальный режим.
- Клавиатурные стрелки активны только когда фокус находится на атласе.
- `aria-live` сообщает выбранный этап.
- `prefers-reduced-motion` отключает декоративные переходы.

## Images

- Все production-изображения лежат локально в `public/assets/`.
- Каждый `EvolutionStage` обязан иметь `image.kind`, `credit`, `license`, `sourceUrl`, `altRu`.
- Текущая версия использует source-backed изображения для всех точек шкалы.
- AI-реконструкции допустимы для будущих недостающих сцен, но должны быть явно помечены как локальные реконструкции.

## Component Sources

- UI-примитивы совместимы с shadcn-подходом и собраны на Radix primitives.
- Визуальное направление вдохновлено подходами Aura/Neuform: строгая токенизация, мягкая глубина, премиальная плотность.
- Иконки: lucide-compatible scientific/navigation set; для будущего расширения можно заменить/добавить icons0 в `src/components/ui`.
