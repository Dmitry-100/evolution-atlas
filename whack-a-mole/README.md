# 🔨 Убей крота кризиса — НЛМК ТОП-100

Мобильная мини-игра-разминка для стратсессии. «Кроты» — это проблемы кризисного
периода, которые лезут наружу: в **производстве**, на **рынке** и в **проектах**.
Игрок за 60 секунд бьёт по ним, набирая очки и серии.

Это **самостоятельный проект**: один файл `index.html` без сборки, зависимостей и
сервера. Работает офлайн в любом мобильном браузере.

## Как играть

- Кроты выскакивают из лунок 3×3 — тапай по ним, пока не спрятались.
- 🏭 красный — производство · 📉 жёлтый — рынок · 🚧 синий — проекты.
- ⚠️ золотой крот = «системная проблема», даёт ×2,5 очков.
- Серия точных ударов наращивает множитель (каждые 4 подряд +1×).
- Промах по пустому полю сбрасывает серию.
- 60 секунд — потом ранг, разбивка по категориям и кнопка «Поделиться».

Есть звук (Web Audio) и вибро-отклик (где поддерживается). Лучший результат
сохраняется в браузере (localStorage).

## Запуск на телефоне

Любой из вариантов:

1. **Просто открыть файл.** Скинуть `index.html` на телефон (мессенджер, AirDrop,
   облако) и открыть в браузере.
2. **GitHub Pages (рекомендуется для зала).** Залить проект в репозиторий →
   Settings → Pages → Deploy from branch → `main` / root. Через минуту будет
   ссылка вида `https://<пользователь>.github.io/<repo>/`. Покажите её QR-кодом —
   участники открывают на своих телефонах.
3. **Локально.** В папке проекта:
   ```bash
   python3 -m http.server 8000
   ```
   и открыть `http://<ip-компьютера>:8000` с телефона в той же сети.

## Вынести в отдельный GitHub-репозиторий

Папка самодостаточна — её содержимое и есть корень будущего репозитория:

```bash
# 1) создайте пустой репозиторий на GitHub (например whack-a-mole-crisis)
# 2) из этой папки:
cd whack-a-mole
git init
git add .
git commit -m "Убей крота кризиса — мини-игра для стратсессии НЛМК ТОП-100"
git branch -M main
git remote add origin git@github.com:<пользователь>/whack-a-mole-crisis.git
git push -u origin main
```

После этого включите GitHub Pages — и игра доступна по ссылке/QR.

## Настройка под свою сессию

Списки проблем легко поменять — они лежат в начале `<script>` в `index.html`
(объекты `PROBLEMS` и `GOLDEN`). Длительность игры — константа `GAME_TIME`,
размер поля — `HOLES`. Брендовые цвета — CSS-переменные в `:root` (`--accent`
синий НЛМК).
